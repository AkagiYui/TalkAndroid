# 现代化安卓应用模板

一个现代化的安卓应用模板，使用 `Kotlin` 与 `Jetpack Compose` 构建。

## 特性

- [ ] Material Design 3 (Material You) 组件
- [ ] i18n 多语言支持(自定义/跟随系统)
- [ ] 深色模式(自定义/跟随系统)
- [ ] 主题色(自定义/跟随系统)
- [ ] Amoled 黑色模式
- [ ] App图标跟随系统主题色
